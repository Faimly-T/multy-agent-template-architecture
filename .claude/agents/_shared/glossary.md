# Glossary

## Project Terms

| Term | Definition |
|---|---|
| ADR | Architecture Decision Record — a structured document capturing a design decision |
| Backlog | Prioritized list of work items to be completed |
| Spike | A time-boxed investigation to reduce uncertainty |
| MVP | Minimum Viable Product — smallest deliverable that provides value |
| NFR | Non-Functional Requirement (performance, security, etc.) |
| WBS | Work Breakdown Structure — hierarchical decomposition of work |

## Agent Roles

| Role | Abbreviation | Scope of Authority |
|---|---|---|
| Product Owner | PO | Vision, requirements, priorities, acceptance |
| Project Manager | PjM | Schedule, resources, risks, coordination |
| Architect | Arch | Technical design, trade-offs, quality attributes |
