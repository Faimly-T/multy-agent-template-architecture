## How to Use This Log

- **Add** questions whenever they arise during a session
- **Update status** at the start of each new session (via Context Rehydration)

## Status Legend

| Status | Meaning |
|---|---|
| `OPEN` | Question is unanswered and potentially blocking |
| `RESOLVED` | Answer was found — resolution is documented |
| `OBSOLETE` | Question is no longer relevant (scope changed, decision superseded it) |
| `DEFERRED` | Intentionally postponed — not needed for current work |
