# Capture Decision

Record a key decision made during a planning session.

## Instructions

For the decision described in: $ARGUMENTS

1. Read `skills/knowledge-capture/SKILL.md` for the capture template
2. Document the decision using the ADR (Architecture Decision Record) format
3. Include: context, options considered, decision rationale, consequences
4. Save to `{paths.decisions}` (resolve from `.claude/settings.json`) with timestamp-based filename
5. Update the decisions INDEX.md with a link to the new record
