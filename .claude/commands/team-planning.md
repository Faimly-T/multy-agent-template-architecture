# Team Planning Session

Orchestrate a collaborative planning session with all three agents.

## Workflow

1. **PO Phase** — Read `agents/product-owner/AGENT.md`. Present the vision and prioritized requirements.
2. **Architect Phase** — Read `agents/architect/AGENT.md`. Propose a technical approach and identify constraints.
3. **PjM Phase** — Read `agents/project-manager/AGENT.md`. Define the execution plan, milestones, and risks.
4. **Alignment** — Synthesize all three perspectives into a unified plan.
