# Prompts Library

Reusable prompt fragments and chains for common agent workflows.

## Naming Convention

`<action>-<subject>.md` — e.g., `elicit-requirements.md`, `review-architecture.md`

## Categories

| Category | Purpose | Example |
|---|---|---|
| `elicit-*` | Gather information | `elicit-requirements.md` |
| `review-*` | Evaluate and critique | `review-architecture.md` |
| `generate-*` | Create artifacts | `generate-user-stories.md` |
| `analyze-*` | Deep analysis | `analyze-risks.md` |
| `summarize-*` | Compress information | `summarize-meeting.md` |
